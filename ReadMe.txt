Лабораторная работа: поиск кратчайшего пути в ориентированном нагруженном графе 
по алгоритму Флойда и по алгоритму Дейкстры, а также их сравнение.

Для демонстрации лучше всего открыть из программы файл 211.xml (вкладка "Граф" кнопка "Открыть")
и перейти на вкладку "Алгоритм".

Вкладка "Матрица" не доведена до ума и может только получать веса.

Кнопка "Пример" лишь загружает массив из текстового файла
